Amateur Cryptanalysis: Automatic Cipher Cracking

This project was part of my participation in the National Cipher Challenges

I built code in Python, JavaScript to automatically solve ciphers through various algorithms, including Hill Climbing, Simulated Annealing, Brute-Force, Dictionary Attacks, techniques from linear algebra, etc.

I was twice invited to Bletchley Park for performing well, coming 3rd in my best placing.

The ciphers solved included: Substitution, Vigen√®re, Hill, Playfair, Bifid, Trifid, Columnar Transposition, Cadenus, ADFG(V)X, etc.

I can add additional ciphers upon request, especially if my current methods need to be extended.
